# 코딩 규칙

> 최종 업데이트: 2026-03-06 01:35:00

## 한글 깨짐 방지

- 아래보다 더 진보적인 신기술이 있다면 반드시 보고하고 규칙을 업데이트할 것
- 현재도 최상의 방법이라면 아래 규칙을 따를 것

### 절대 규칙
1. 한글 문자열은 유니코드 이스케이프(\uXXXX) 사용 금지. 반드시 한글 직접 작성
2. 한글 포함 파일 수정 시 Edit tool 기본 사용 (속도 우선)
3. 한글 수정 후 반드시 Read tool로 해당 부분 한글 정상 출력 확인
4. 한글 깨짐 발견 시 Write tool로 즉시 재저장 (보험)
5. PowerShell 리다이렉션 사용 금지
6. 파일 인코딩은 UTF-8 유지

## 텍스트 파일 이모지 제한

### 절대 규칙
1. 확장자 .txt 파일과 .md 파일에는 사용자가 요청하지 않는 한 이모지를 파일에 쓰지 않음
